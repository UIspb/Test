# ArtPrints - Сайт для продажи арт-принтов

Современный веб-сайт для продажи художественных принтов, созданный в стиле шаблона Jesper с использованием HTML, CSS и JavaScript.

## 🎨 Особенности

- **Современный дизайн** в стиле Jesper с элегантными анимациями
- **Адаптивная верстка** для всех устройств
- **Плавные анимации** с использованием GSAP
- **Интерактивная галерея** с фильтрацией по категориям
- **Эффекты параллакса** и scroll-анимации
- **Мобильное меню** с плавными переходами
- **Загрузочный экран** с анимацией
- **Курсор-следователь** для desktop версии

## 🚀 Технологии

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Google Fonts (Inter)
- Unsplash API для изображений

## 🛠 Установка и запуск

1. Склонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере
3. Для разработки рекомендуется использовать локальный сервер

## 🎨 Дизайн-система

### Цвета
- Основной: #333333
- Фон: #fafafa
- Белый: #ffffff
- Серый: #666666

### Типографика
- Шрифт: Inter (Google Fonts)
- Размеры: от 12px до 80px
- Веса: 300, 400, 500, 600, 700

### Анимации
- Easing: power2.out, cubic-bezier(0.25, 0.46, 0.45, 0.94)
- Длительность: 0.3s - 2s
- Задержки: stagger эффекты
